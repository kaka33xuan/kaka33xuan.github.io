Markdown常用语法学习笔记

发表于2024-06-07|更新于2024-06-07|其他
|阅读量:

本笔记来自Markdown 官方教程

Markdown标题语法

“ # ”一级标题；

“ ## ”二级标题；

“ ### ”三级标题；

…

以此类推，井号后添加空格。

Markdown强调语法

粗体：“文本” 文本前后添加双星号；

斜体：“文本“ 文本前后添加单星号；

粗体和斜体:”文本“ 文本前后添加三个星号；

Markdown换行与段落

换行要两个或多个空格，然后按回车键；段落直接按回车键。

Markdown引用语法

要创建块引用（字体颜色变浅），请在段落前添加一个 > 符号；

多个段落添加多个 > 符号；

块引用可以嵌套。在要嵌套的段落前添加一个 >> 符号。

Markdown列表语法

  1. 创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。
  2. 创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。
  3. 在列表中嵌套其他元素时，段落和引用缩进四个空格，代码块缩进八个空格。

Markdow代码语法

要将单词或短语表示为代码，前后添加反引号 (`) ；

若原本包含反引号，可以前后添加反双引号(``)；

代码块前后三个反引号（```）或三个波浪号（~ ~ ~）。

Markdown 分隔线语法

要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (---) 或下划线 (___) ，并且不能包含其他内容。

Markdown 链接语法

  1. 链接名称放在中括号内，链接地址放在后面的括号中，链接title可选。超链接Markdown语法代码：[超链接显示名](超链接地址 "超链接title")；
  2. 邮箱地址，给其前后添加尖括号<>；
  3. 链接前后也可以增加星号表示强调或粗体；

Markdown 图片语法

  1. 使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。插入图片Markdown语法代码：![图片alt](图片链接 "图片title")。
  2. 给图片增加链接，请将图像的Markdown 括在方括号中，然后将链接添加在圆括号中。

Markdown 表格

  1. 要添加表，请使用三个或多个连字符（---）创建每列的标题，并使用管道（|）分隔每列。您可以选择在表的任一端添加管道。

  1. 通过在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐到左侧，右侧或中心。

Markdown 脚注

前后添加方括号，方括号中加^和名称，比如[ ^1 ]；

Markdown 标题编号（可忽略）

  1. 要添加自定义标题ID，请在与标题相同的行上用大括号括起该自定义ID。如
  1. 通过创建带有数字符号（#）和自定义标题ID的标准链接，可以链接到文件中具有自定义ID的标题。其他网站可以通过将自定义标题ID添加到网页的完整URL（例如[Heading IDs](https://markdown.com.cn/extended-syntax/heading-ids.html#headid)）来链接到标题。

Markdown 定义列表

要创建定义列表，请在第一行上键入术语。在下一行，键入一个冒号，后跟一个空格和定义。

markdown 删除线

要删除单词，请在单词前后使用两个波浪号~~。

Markdown 任务列表语法

要创建任务列表，请在任务列表项之前添加破折号-和方括号[ ]，并在[ ]前面加上空格。要选择一个复选框，请在方括号[x]之间添加 x

Markdown 使用 Emoji 表情

可以从Emojipedia复制表情符号并将其粘贴到文档中。
